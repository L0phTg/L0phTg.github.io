    总结了一下自己的vim配置和zsh配置

    https://github.com/robbyrussell/oh-my-zsh

    vim配置

    1. 当你使用vim时, 可以在~/.vimrc中编写你的vim配置文件.

    2. 当你使用neovim时, 可以在~/.config/nvim/init.vim中编写你的vim配置文件.

    插件

    1. 插件管理vim-plug
    2. tagbar
    3. airline
    4. vimfilter
    5. vim-bufabline buffer窗口
    6. fzf和ag 搜索
    7. 括号补全'Raimondi/delimitMate’
    8. include补全 neoinclude
    9. 语法补全jedi-vim(python)

    vim-plug vim插件管理工具

    README: https://github.com/junegunn/vim-plug.

    1. 安装

    a. 当你使用vim时.

    curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim

    b. 当你使用`neovim时.

    curl -fLo ~/.local/share/nvim/site/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
    1. 使用

    a. 可以使用git链接

    Plug 'https://github.com/junegunn/vim-github-dashboard.git'

    b. 使用gitUserName/Project.

    Plug 'junegunn/vim-easy-align'

    更多可以参考文档.

    1. 安装插件

    只是把要安装的插件写在配置文件中是不够的, 还需要安装.

    写好配置文件后, 重新打开vim, 运行PlugInstall.

    1. 例子:(我的部分配置)
    call plug#begin()

Plug 'NLKNguyen/papercolor-theme'
Plug 'vim-airline/vim-airline'
Plug 'vim-airline/vim-airline-themes'
Plug 'Shougo/denite.nvim'

Plug '~/.fzf'

call plug#end()

    tagbar

    README: https://github.com/majutsushi/tagbar.

    1. 安装ctags
    $sudo apt install ctags
    1. vim中配置tagbar
    call plug#begin()
Plug 'majutsushi/tagbar'
call plug#end()

let g:tagbar_width=30
let g:tagbar_left=1

"映射F2快捷键为打开tagbar
nnoremap <F2> :TagbarToggle <CR>

    fzf 和 ag

    1. 下载安装fzf
    git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf
~/.fzf/install
    1. vim 中配置fzf

    插件使用vim-plug

    Plug '~/.fzf'
    1. ag安装

    ubuntu

    $sudo apt-get install silversearcher-ag
    1. 快捷键映射

    在我的vim中, 映射为了space

    noremap <leader>sff :Files<CR>
noremap <leader>ssf :Lines<CR>
noremap <leader>sbf :BLines<CR> 
noremap <leader>ss  :Ag<CR>

noremap <leader>gls :GFiles<CR>
noremap <leader>gss :GFiles?<CR>   

noremap <leader>btag :Tags<CR>
noremap <leader>stag :BTags<CR>
    1. 使用

    :BTags 可以在当前目录下build tags, 前提系统已经ctags(apt install ctags)

    :Tags 可以搜索tag. 在搜索前, 要先build tags. (可以自行阅读fzf doc, 应该可以在vim启动时自动build tags).

    括号补全'Raimondi/delimitMate’

    call plug#begin()
Plug 'Raimondi/delimitMate'       " 补全, \" \( 等.
call plug#end()

    c/c++/llvm头文件补全 neoinclude

    头文件补全需要设置头文件path, (注意, 是vim配置文件中的path, 而不是系统环境变量的path)

    我使用的是nvim, 配置文件是~/config/nvim/init.vim

    " ~/.config/nvim/init.vim

set path=.,/usr/include/c++/5,/usr/include,/home/l0phtg/tools/llvm-project/llvm/include

call plug#begin()
Plug 'Shougo/neoinclude.vim'
call plug#end()
